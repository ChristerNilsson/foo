# Nim

* Click one or many times on ONE button!
* Then click on Go!
* Player taking the last stick wins!

[Wikipedia](https://en.wikipedia.org/wiki/Nim)

[The first computer game (1940)](https://en.wikipedia.org/wiki/Nimatron)
